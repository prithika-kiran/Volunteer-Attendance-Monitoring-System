# AI-Powered Volunteer Attendance Monitoring System

## Project Overview

The AI-Powered Volunteer Attendance Monitoring System is a computer vision application developed using Python and OpenCV. The system detects faces in real time through a webcam and automatically records volunteer attendance with date and time information.

This project was designed to help organizations and NGOs automate attendance tracking during events, workshops, and volunteer activities.

---

## Features

* Real-time face detection using Haar Cascade Classifier
* Automatic attendance logging
* Attendance records stored in CSV format
* Face count display
* Live webcam feed
* Timestamp generation for attendance records

---

## Technologies Used

* Python
* OpenCV
* Haar Cascade Classifier
* CSV File Handling
* Datetime Module

---

## Project Workflow

1. User enters volunteer name.
2. Webcam starts capturing video.
3. Frames are converted to grayscale.
4. Haar Cascade detects faces in the frame.
5. Attendance is automatically recorded when a face is detected.
6. Date and time are saved in a CSV file.
7. Face count is displayed on the screen.
